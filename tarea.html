ACTIVIDAD 1. Gestión del DOM y navegación entre nodos

Implementa en HTML una página Web que contenga tres inputs (nombre, apellido y edad) y un botón de añadir. Además, debe contener una tabla con los mismos encabezados.  Al pulsar el botón de añadir, se debe añadir una nueva fila a una tabla.

  • El botón de añadir añadirá al final de la tabla una nueva fila con los datos que contuvieran sus inputs. Posteriormente, se debe borrar el contenido de los inputs.
  • El botón solamente estará activo si se ha introducido un valor valido en los tres inputs. También se desactivará si se está editando una fila.
  • Estas filas deben añadirse utilizando el sistema de nodos.

Adicionalmente, cada fila de la tabla debe contar con botones para las siguientes operaciones: Editar, Aceptar, Eliminar, Subir y Bajar.  Estos botones deben ser enlazados con funciones (eventos) cuando son insertados en la tabla. En concreto, las acciones a añadir son:

  • Si se pulsa el botón de editar de una fila, los párrafos que contienen sus datos se convertirán en inputs.
  • Solo se deberá poder editar una fila simultáneamente.
  • Si se pulsa el botón de aceptar (solo disponible si en una columna se ha pulsado anteriormente el botón de enviar), se convertirán los inputs de nuevo en párrafos con los valores actuales.
  • Si se pulsa el botón de borrar se eliminará su fila correspondiente.
  • Si se pulsa el botón de subir se sube la fila actual a la fila superior. No debe poder utilizarse en la primera fila.
  • Si se pulsa el botón de bajar se baja la fila actual a la fila inferior. No debe poder utilizarse en la última fila.

Por último, El HTML de la tabla debe estar vacío (salvo por el encabezado), su estilo ser dado mediante CSS y su contenido añadirse mediante código JavaScript. Desde la función init(), se debe añadir al menos 4 filas a la tabla mediante JavaScript.

 ACTIVIDAD 3. Página Web de CV con React y Router

Crea una web de tu curriculum vitae (CV) haciendo uso de React y React Boostrap (u otra de las librerías de componentes de React como MaterialUI). Aquí puedes ver un ejemplo.

Esta debe tener varias páginas, todas compartiendo un header con un menú de navegación y un footer (Componentes: <MyHeader> <MyFooter>). Estas deben ser las siguientes (para utilizar varias páginas hay que utilizar React Router):

  • Página de aterrizaje (Componente: <LandingPage>). Debe mostrar tu nombre, una fotografía (puede ser la de tu perfil de moodle), tus perfiles (github, linkedin, etc.), un par de frase describiéndote y dos o tres cualidades/habilidades que te gustaría destacar. Además, se puede hacer que al hacer scroll hacia abajo vaya apareciendo más información que quieras destacar o poner un enlace para descargar un PDF con tu CV.
  • Página de proyectos (Componente: <PortafolioPage>). Varias imágenes de proyectos realizados a lo largo del ciclo, con título y breve descripción. Aquí suele ser adecuado poner enlaces a github.

  • Página de experiencia académica y laboral (Componente: <ExperiencePage>).
  • Si se pone cualquier URL que sea diferente a las anteriores, debe reconducir directamente a la página de aterrizaje.

Se puede modificar ligeramente la estructura de página (por ejemplo, poniendo los iconos de los perfiles en el header o footer) para adaptarla a tus necesidades. Siempre que se mantenga el próposito de la Web CV, esté hecha con React y haya al menos dos páginas diferentes.
